# CineVerse
just a little site for uni


CineVerse este un website dedicat pasionaților de filme și seriale, realizat în HTML și CSS. Proiectul funcționează ca un ghid online care reunește recomandări, trailere, categorii variate și resurse vizuale, într-o interfață organizată și ușor de parcurs.

Platforma include secțiuni pentru filme, seriale și documentare, recomandări tematice, galerie de imagini, trailere integrate, linkuri externe utile și resurse descărcabile. De asemenea, proiectul urmărește folosirea corectă a elementelor semantice HTML și a componentelor cerute într-un site modern.
